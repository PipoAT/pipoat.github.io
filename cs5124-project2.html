Live Application

Use of AI and Collaboration

AI tools including ChatGPT and Copilot were utilized in providing customization for the world map to ensure that the map is constrained properly. Also, AI was used to debug and address any bugs are they occured while in development.


Overview

This application displays earthquakes that have occured from March 2024 - March 2025 throughout the world.


Data Sources

The datasets used in this application include:

  • USDA Atlas of Rural and Small-Town America (Socioeconomic Data).
  • US Heart and Stroke Atlas (Health and Healthcare Data).

Technology Stack

  • D3.js: Data visualization library.
  • Leaflet Handles the World Map to display data
  • JavaScript: Handles interactivity.
  • HTML & CSS: Provides structure and styling.

Process

  • Libraries used: D3.js, Leaflet, JavaScript, HTML, CSS.
  • Additional Frameworks: Bootstrap
  • Code structure: The application is divided into modules for data loading, visualization, and interactivity.
  • Deployment: Hosted on Vercel, with source code available on GitHub.

Challenges
